# Contributing

## Development

- `make schemas-fetch` — download OCA JSON schemas
- `make codegen` — regenerate message types from schemas
- `make test` / `make test-race` — run tests
- `make lint` — run golangci-lint

## Generated code

Files with `// Code generated by gocpp codegen. DO NOT EDIT.` are produced by
`internal/codegen`. Never edit them by hand. Change the schema or profile YAML
and re-run `make codegen`. CI fails if generated output drifts from committed.
